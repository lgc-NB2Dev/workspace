# Workspace

开发 NoneBot2 插件的工作区

## 开发

### 克隆 & 更新

可以使用以下命令来克隆本工作区进行插件开发

```bash
git clone --recurse-submodules --depth=1 https://github.com/lgc-NB2Dev/workspace
```

可以使用下面的命令更新所有子模块

```bash
git submodule foreach git pull
```

### 安装工作区开发依赖

使用 `pdm` 来安装依赖

```bash
pdm install --group all --no-self
```
